# ha-kakaonavi
카카오모빌리티 디벨로퍼스의 길찾기 API (https://developers.kakaomobility.com/docs/navi-api/start/)를 사용하여, HomeAssistant에서 출발지와 경유지, 목적지까지의 예상 도착시간을 알 수 있다.

### 제공기능
1. 현재 ETA
2. 30분 후 ETA
3. 현재와 미래 ETA 차이
4. 거리, 택시 요금, 통행료 정보
5. 최적 출발 시간 찾기 서비스
